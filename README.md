# Jacob Hetzner — Künstler-Website

Persönliche Website von Jacob Hetzner — Schauspieler, Sänger, Komponist.

## Lokal ansehen

`index.html` einfach im Browser öffnen (Doppelklick).

## Auf GitHub pushen

```bash
cd pfad/zum/ordner
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/jacobhetzner.git
git push -u origin main
```

## Hosten (empfohlen: Vercel)

1. Account auf [vercel.com](https://vercel.com) anlegen
2. GitHub-Repo verbinden
3. Deploy — fertig, kostenlos und mit automatischem HTTPS

## Domain verbinden

In Vercel unter "Domains" → eigene Domain eintragen (z.B. `jacobhetzner.de`).

## Mit KI weiterarbeiten

Der Ordner ist für Gemini CLI vorbereitet. In Gemini CLI Desktop den Ordner öffnen — die `GEMINI.md` wird automatisch gelesen.

## Struktur

```
/
```
